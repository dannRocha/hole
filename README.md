# Hole

## Ferramentas:
- :pushpin: Java 11
- :pushpin: Maven
- :pushpin: MariaDB 10.7
## Proposta
API com o objetivo de registrar buracos nas avenidas e estradas das cidades

## :cd: Iniciar

Na linha de comando:
Use o ```mvnw``` para ambiente Linux e ```mvnw.cmd``` para Windows
```
./mvnw spring-boot:run
```
